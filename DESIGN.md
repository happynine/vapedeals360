# DESIGN.md

## 气质与意象

**关键词**：暗夜霓虹、科技感、价格猎手、精致雾化

**具象场景**：深夜的电子烟体验店，黑色大理石展台上陈列着金属质感的设备，柔和的渐变灯光（紫→青）映在哑光表面，空气中飘着淡淡的水果雾气。顾客扫码查看各渠道价格，屏幕上跳动的数字像霓虹招牌一样吸引目光。

## 视觉策略

- 整体暗色主题，营造高端科技感，与电子烟品类的潮流属性呼应
- 卡片采用深灰底 + 微光边框，模拟金属拉丝质感
- 产品主图在暗底上更突出，类似橱窗展示效果
- 降价标签用亮色高对比突出，像霓虹折扣牌

## 配色方案

| 用途 | 颜色 | 意象来源 |
|------|------|----------|
| 背景 | #0f0f13 (极深灰黑) | 暗夜店铺墙面 |
| 卡片底 | #1a1a24 (深灰蓝) | 大理石展台 |
| 主强调 | #7c3aed (亮紫) | 霓虹灯光 |
| 次强调 | #06b6d4 (青蓝) | 雾化蒸汽 |
| 降价标签 | #ef4444 (红) | 限时折扣牌 |
| 现价 | #22c55e (绿) | 省钱信号 |
| 原价 | #6b7280 (灰) | 划线废弃价 |
| 正文 | #e5e7eb (浅灰白) | 夜间阅读灯光 |

## 字体排版

- 英文：Inter（清晰现代，适合价格数字）
- 中文回退：PingFang SC / Microsoft YaHei
- 价格数字：tabular-nums，等宽对齐
- 产品名：text-lg font-semibold
- 商城名：text-sm font-medium

## 动效与交互

- 卡片 hover：微上浮 + 边框发光（紫色光晕）
- 降价标签：微脉冲动画（scale 1→1.03 循环）
- 页面加载：卡片依次淡入（stagger animation）
- 商城按钮 hover：渐变背景 shift

## 页面结构

### 首页
- 顶部导航：Logo + 分类筛选 + 搜索
- Hero 区域：今日降价精选轮播
- 分类 Tab 筛选条
- 产品卡片网格（3-4列）
- 卡片内容：产品图 | 产品名 | 原价→现价 | 降价百分比标签 | 多商城价格列表 + 跳转按钮

### 详情页
- 产品大图 + 图片画廊
- 产品名称 + 描述
- 规格参数表
- 价格对比表（各商城价格 + 链接按钮）
- 相关推荐

### 后台管理
- 左侧导航栏
- 多语言切换标签
- 表格式 CRUD 操作

## 设计禁忌

- 禁止使用浅色/白色背景作为主色调（与电子烟品类调性不符）
- 禁止使用彩色渐变作为大面积背景（杂乱，分散产品注意力）
- 禁止卡片使用纯黑底（缺乏层次感）
- 降价信息不可被弱化展示（核心价值）
- 禁止使用卡通风格图标（与高端定位冲突）

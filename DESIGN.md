# DESIGN.md

## 气质与意象

**关键词**：白底清爽、紫调强调、价格猎手、精致雾化

**具象场景**：明亮的现代电子烟展示厅，白色展台上陈列着金属质感的设备，紫色渐变灯光点缀其间。顶部和底部深色框架（导航+页脚）形成视觉包围，中间内容区域白底清爽，产品图片和价格信息一目了然。

## 视觉策略

- 内容区域白底为主，产品图片在白底上更清晰突出
- 顶部导航和底部页脚采用深色（#0a0a0e），形成视觉框架
- 卡片采用白底 + 浅灰边框，干净利落
- 降价标签用亮色高对比突出
- 紫色作为品牌强调色贯穿全站

## 字体排版

- 英文：Inter（清晰现代，适合价格数字）
- 中文回退：PingFang SC / Microsoft YaHei
- 价格数字：tabular-nums，等宽对齐
- 产品名：text-lg font-semibold
- 商城名：text-sm font-medium

## 动效与交互

- 卡片 hover：微上浮 + 边框变紫色
- 降价标签：微脉冲动画（scale 1→1.03 循环）
- 页面加载：卡片依次淡入（stagger animation）
- 商城按钮 hover：背景色加深

## 页面结构

### 首页
- 顶部深色导航：Logo + Tab导航(Vape Deals/Best Vapes/News) + 排序按钮
- Banner轮播区域
- 分类筛选条
- 产品卡片网格（白底卡片）
- 深色Footer三列布局

### 内容页(Best Vapes/News)
- 深色导航栏
- 白底内容列表/详情
- 深色Footer

### 静态页(About/Privacy/Contact)
- 深色导航栏
- 白底内容
- 深色Footer

### 详情页
- 产品大图 + 图片画廊
- 价格对比表（白底卡片）
- 规格参数表
- 深色Footer + FDA/NIXODINE Disclaimer

### 后台管理
- 左侧导航栏
- 多语言切换标签
- 表格式 CRUD 操作
- 富文本编辑器(react-quill)

## 设计禁忌

- 禁止在内容区域使用深色背景（与白底设计冲突）
- 禁止使用彩色渐变作为大面积背景（杂乱，分散产品注意力）
- 卡片不可无边框（白底白卡会融为一体）
- 降价信息不可被弱化展示（核心价值）
- 禁止使用卡通风格图标（与高端定位冲突）
